La aplicación consta de tres tablas (lugar,categoria,plato).
La tabla lugar tiene los campos id, nombre y estrellaB. La tabla categoria tiene id y nombre. 
La tabla plato tiene id, nombre, precio, estrellaP,categoriaId y barId.
Hay una relación entre las tablas lugar(id) y plato (barId), y otra relacion entre categoria (id) y plato (categoriaId). 
Se puede añadir, actualizar y eliminar en las tres tablas.
